Import bài viết từ file Word

MVP: 1 bài / lần, hỗ trợ .docx, lưu file Word gốc và .zip ảnh phụ.

Hủy

Cách hoạt động

  • Dòng đầu tiên trong file Word được dùng làm tiêu đề nếu bạn không nhập tiêu đề thay thế.
  • File Word gốc sẽ được lưu vào bài tin để có thể tải lại sau này trong mục tin tức.
  • Ảnh nhúng trong file Word sẽ được đưa vào nội dung và ảnh đầu tiên sẽ làm thumbnail.
  • Nếu có ảnh rời, bạn nén thành một file .zip và upload vào ô ảnh phụ.
  • Nếu bật ghi đè, portal sẽ cập nhật bài cũ theo tiêu đề hoặc slug thay vì tạo bản sao.
